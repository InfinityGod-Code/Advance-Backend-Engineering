## Project Configuration
The project demonstrate the core concepts like concurrency, locking strategies and deadlock handling.

#### Project Installation and Running Commands
- install UV package manager if not installed in your system. You can install that from here : [link](https://docs.astral.sh/uv/getting-started/installation/)
- run FastAPI demontration application from these following commands :
  - uvicorn main:app --reload
  - alembic is used for database migrations
